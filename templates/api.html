Headless API

Script the same staged docking-preparation workflow used by the browser interface.

PrepServer includes versioned API endpoints for teams that want to automate workspace creation, receptor ingestion, docking-center resolution, ligand upload, receptor preparation, and package generation.

Health check

GET /api/v1/health

Workspaces

POST /api/v1/workspaces
GET /api/v1/workspaces/<jobname>

Receptors

POST /api/v1/workspaces/<jobname>/receptors/upload
POST /api/v1/workspaces/<jobname>/receptors/fetch

Centers

POST /api/v1/workspaces/<jobname>/centers/resolve
POST /api/v1/workspaces/<jobname>/centers/save

Ligands

POST /api/v1/workspaces/<jobname>/ligands/upload
GET /api/v1/workspaces/<jobname>/ligands

Build artifacts

POST /api/v1/workspaces/<jobname>/build
GET /api/v1/workspaces/<jobname>/artifacts

Staged workflow by design

The current API favors staged operations over a single opaque one-call workflow so setup failures remain visible and easier to debug.
